# Entrega mínima a Codex

## Qué hacer

1. Abre **una sesión nueva de Codex**.
2. Usa intensidad **Alta**.
3. Sube únicamente este ZIP.
4. Pega este mensaje:

```text
Trabaja en un repositorio nuevo y ejecuta exactamente
CODEX_PRIMARY_BUILD_PROMPT.md.

Lee primero 00_START_CODEX.md y HANDOFF_MANIFEST.md.

No uses código previo de Work, no solicites API key, no hagas llamadas
a la API y no abras subhilos. Construye el núcleo completo en esta misma
sesión, ejecuta las pruebas y detente al llegar a la condición de terminado.
```

## No entregar a Codex

No subas:

- el ZIP funcional construido en Work;
- código de Atlas Vivo;
- capturas del sitio;
- archivos de Historia Madre;
- URLs o IDs privados;
- credenciales;
- otros paquetes anteriores.

Este paquete contiene todo el insumo autorizado.
